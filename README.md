# Vert.x React Demo

## Pre-requisites

- JDK 8


## Running the module

In dev mode:

    mvn clean compile vertx:runMod

Or build the module (below) and run it on your local Vert.x 2.1RC3.


## Building the module

    mvn clean package


## Profiting!

After starting the module, browse: http://localhost:8080/
